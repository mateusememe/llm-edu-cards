# deep-learning-llm-educational-cards
Sistema educacional interativo que utiliza modelos de linguagem de grande escala (LLMs) para gerar conteúdo educacional estruturado em formato de cards, desenvolvido como trabalho prático da disciplina de Aprendizado Profundo do Programa de Pós-Graduação em Ciência da Computação da UNESP, ministrada pelo Prof. Dr. Denis Henrique Pinheiro Salvadeo.
